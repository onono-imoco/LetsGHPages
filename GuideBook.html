アカウントの存在確認

無ければ作るべし

SSHキー登録

GitHubホーム > 右上アイコン > Settings > SSH and GPG keys できてなければ登録すべし

鍵の生成

下記の通り進めれば作れる Macで公開鍵と秘密鍵を生成する方法

GitHubに登録

公開鍵をGitHubに登録する

Node.jsの存在確認

node -v

無ければインストールすべし

npmの存在確認

npm -v

無ければインストールすべし

yarnの存在確認

yarn -v

無ければインストールすべし

gitの存在確認

git -v

追加で、gitのデフォルトブランチが"main"になっていることを確認すべし。 なってなければデフォルトを"main"にすること。

バージョン管理する

git init
git commit -m 'first commit'

GitHubにpushする

git remote add origin https://github.com/<ACCOUNT_NAME>/<REMOTE_REPO_NAME>.git
git remote set-url origin git@github.com/<ACCOUNT_NAME>/<REMOTE_REPO_NAME>.git
git push origin main

あとは流れに身を任せて
